# KaseiCoin
A new cryptocurrency for the first human colonization of Mars using a crowdsale to convert the earthling money to KaseyCoin.

## Background

Selected by the Martian Aerospace Agency, I was chosen to lead a project to develop a monetary system for the first human colony. Using blockchain technology, I defined a new cryptocurrency, named KaseiCoin. (Kasei means Mars in Japanese.)

KaseiCoin will be a fungible token that’s ERC-20 compliant. A crowdsale will be launched to allow people who are moving to Mars to convert their earthling money to KaseiCoin.
